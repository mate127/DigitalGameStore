Kod aplikacije može se otvoriti u razvojnom okruženju Visual Studio 2022. Kroz razvojno okruženje pokreće se web aplikacija.
Za pokrenuti aplikaciju potrebno je napraviti bazu podataka. Skripta za kreiranje baze i inicijalno punjenje podacima dana je u prilogu DGS-DZ03prilozi.zip. Nakon toga potrebno je u appsettings.json postaviti connection string na vlastitu bazu podataka.
Nakon pokretanja web aplikacije trebao bi se otvoriti prozor u web pregledniku pomoću kojeg se navigira kroz aplikaciju.
